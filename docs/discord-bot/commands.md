## Global Commands <Badge type="warning" text="NEW"/>

| :----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| invite       | Senchabot Discord bot invite url.                                                                                                                                                                                                    |
